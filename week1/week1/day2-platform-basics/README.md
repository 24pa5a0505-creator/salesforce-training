## 🛠 Key Technical Concepts

### Salesforce App
An App in Salesforce is a collection of related tabs, objects, and functionalities grouped together for a specific business purpose.

### Object
Objects are database tables used to store records and data inside Salesforce.

### Tab
Tabs are user interface components used to access objects, records, dashboards, and applications.

### Multi-tenant Architecture
Salesforce uses a multi-tenant architecture where multiple organizations share the same infrastructure securely.

### Configuration vs Coding

#### Configuration
Customization using clicks without writing code.
Examples:
- Creating fields
- Validation rules

#### Coding
Customization using Apex and Lightning components.
Examples:
- Custom business logic
- API integrations
